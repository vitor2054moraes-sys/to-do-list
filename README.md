# 📝 To-Do List

Uma aplicação simples de lista de tarefas feita com HTML, CSS e JavaScript puro.

## 🚀 Funcionalidades

- ➕ Adicionar novas tarefas
- ✅ Marcar tarefas como concluídas (fica riscada)
- 🗑️ Remover tarefas da lista

## 🛠️ Tecnologias

- HTML5
- CSS3
- JavaScript
